HTML & CSS Quick-Reference Guide

Basic HTML Tags

The following are some essential HTML tags:

  • h1 - Heading of your webpage
  • p - Paragraph
  • a - Anchor a link in your webpage
  • img - Add an Image
  • ul - Unordered List
  • ol - Ordered List
  • li - List Item
  • link - Link stylesheet
  • Body - includes the main content of your webpage
  • title - set a title for your webpage

CSS Properties

Here are some common CSS properties:

  • color
  • font-size
  • background-color
  • padding
  • margin
  • border
  • text-align
  • display
